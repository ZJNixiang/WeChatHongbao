# iOS版微信抢红包插件

微信后台自动抢红包，不需要任何的手动操作；开发这个插件初衷是为了研究在微信群里有些用户在抢到红包之前就能知道 **红包金额**；提前知道金额后就能卖给一些非法盈利和赌博人群，很多人因此被骗很多钱财，借此希望大家不要误入歧途。

**本插件只用于学习目的，请勿使用于别的用途**

## 使用
   在越狱开发环境完整情况下：

- make package install
- 打开‘微信’
